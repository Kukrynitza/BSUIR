' -----------------------
'  ПАТТЕРН: Цепочка Обязанностей (фиксированный порядок)
' -----------------------
abstract class Handler {
- next: Handler
+ setNext(next: Handler)
+ handle(контекст)
}

class HandlerБКИ <<handler>> {
+ handle(контекст)
}
class HandlerБДБанк <<handler>> {
+ handle(контекст)
}
class HandlerФинАнализ <<handler>> {
+ handle(контекст)
}
class HandlerФормированиеРешения <<handler>> {
+ handle(контекст)
}

Handler <|-- HandlerБКИ
Handler <|-- HandlerБДБанк
Handler <|-- HandlerФинАнализ
Handler <|-- HandlerФормированиеРешения

' Жёсткая цепочка (стрелки показывают порядок обработки)
HandlerБКИ --> HandlerБДБанк
HandlerБДБанк --> HandlerФинАнализ
HandlerФинАнализ --> HandlerФормированиеРешения

' Контроллер создаёт/инициирует цепочку
КредитныйКонтроллер ..> HandlerБКИ : инициализирует
note right of HandlerБКИ
Фиксированный порядок обработки:
1) БКИ
2) БД Банка (чёрный список)
3) Финансовый анализ
4) Формирование решения
end note
